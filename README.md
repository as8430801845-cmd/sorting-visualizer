# 🚀 Sorting Algorithm Visualizer

An interactive web-based application to visualize and understand how different sorting algorithms work in real-time.

## 📌 Features

- 🔢 Visualizes multiple sorting algorithms:
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort
- 🎮 Dynamic controls:
  - Adjust array size
  - Control sorting speed
  - Select different algorithms
- 🎨 Real-time animations for comparisons and swaps
- 📱 Responsive and user-friendly interface

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)
